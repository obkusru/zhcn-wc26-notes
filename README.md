# zhcn-wc26-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容以笔记、工具页面、静态演示等形式为主，不针对任何特定域名或网站。

## 项目简介

本仓库主要用于存放各种独立编写的 HTML 文件。这些文件可以是单页工具、信息展示页、学习笔记页面或实验性静态页面。仓库本身不提供统一入口，每个 HTML 文件即为一个独立单元。

## 目录说明

仓库根目录下直接存放 HTML 文件，或按功能/主题划分为子目录。当前主要目录结构如下：

- `pages/` —— 存放主要归档页面
- `tools/` —— 存放单页工具类 HTML
- `notes/` —— 存放笔记或文档类 HTML
- `assets/` —— 公共资源文件（CSS、JS、图片等）

目录结构可能随时间调整，请以实际仓库文件为准。

## 页面归档说明

每个 HTML 文件均为自包含页面，内部已包含必要的样式和脚本。部分页面可能引用 `assets/` 目录下的公共资源。页面内容以实用、学习或记录为目的，不包含商业推广或诱导性内容。

## 使用方式

你可以直接通过 GitHub Pages 或任何静态服务器访问本仓库中的 HTML 文件。推荐方式：

1. 克隆仓库到本地
2. 在浏览器中直接打开对应 HTML 文件
3. 或部署到任意静态托管服务

## 维护说明

本仓库由个人维护，不定期更新。如果你发现页面内容有误或需要改进，欢迎提交 Issue 或 Pull Request。对于新增页面，请保持风格一致，并尽量使用自包含的 HTML 文件。
